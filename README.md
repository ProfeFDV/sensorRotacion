## Descripción 
 Mostrar cara sorprendido cuando se agita el Microbit ".<br>
 Cuando el microbit no se agita, muestra una fecha rotando.
 
## Usar como extensión

Este repositorio puede ser añadido como una **extensión** en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **New Project**
* haga clic en **Extensiones** en el menú del engranaje
* buscar **https://github.com/profefdv/sensoragitacion** e importar

## Edita este proyecto ![Insignia de estado de compilación](https://github.com/profefdv/sensoragitacion/workflows/MakeCode/badge.svg)

Para editar este repositorio en MakeCode.

* abra [https://makecode.microbit.org/](https://makecode.microbit.org/)
* haga clic en **Import** luego haga clic en **Import URL**
* pegue **https://github.com/profefdv/sensoragitacion** y haga clic en importar
